# Data structure of Protocol Designer JSON in aspect of data handling
Not atempting to generate PY2JSON, but the JSON file will be the scaffold of collecting OT-2 control info, in CSV form, generated by dna-origami-db or by hand.
The nested level title represents nested hash key. The bullet list tells the data structure of lower layer, from which column of CSV file the date is copied from or the rule of data processing.
